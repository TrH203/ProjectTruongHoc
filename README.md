# Simple School Application Description

This is a simple application that describes school functions such as:

1. **Login Rights**
   - There are 3 login rights, each with different functions.

2. **Administrator Functions**
   - Administrators have the ability to create accounts for students and teachers, but first, there must be information in the system created by the administrator.
   - Administrators can also create classes and add students to those classes for study.

3. **Teacher Functions**
   - Teachers can view the classes they are teaching.
   - Teachers can create assignments for students in their classes.

4. **Student Rights**
   - Students can view information.
   - Students can view their current classes.
   - Students can view their assignments.
